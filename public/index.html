USER INFORMATION
HTTP Method Path Description
GET /users Gets all users
GET /users/:id Gets a single user by id
POST /users/register Adds a new user
PATCH /users/update/:id Edit a user's information by id
DELETE /users/delete/:id Removes a user by id
PRODUCT INFORMATION
GET /products Gets all products
GET /products/:id Gets a single product by id
POST /products/addProduct Adds a new product
PATCH /products/update/:id Edit a product's information by id
DELETE /products/delete/:id Removes a product by id
